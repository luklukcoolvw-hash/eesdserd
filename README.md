Lotto-Simulator

Dieses kleine Python-Skript simuliert ein Lotto (6 aus 49).

Benutzung:

- Direkt ausführen:

  python lotto_simulator.py

- Funktionen (für Import/Tests):
  - `generate_ticket(numbers=None)`
  - `draw()`
  - `check_ticket(ticket, drawn)`
  - `simulate(n_draws, ticket=None)`
